# Nala brand source of truth

This document reconciles the production brand system with the current Nala product.

## Product definition

Nala is a South African work-experience engine that turns real small-business needs into safe, paid starter tasks and turns verified completion into portable proof of employability.

Core loop:

`business need -> structured task -> readiness practice -> guided paid work -> business verification -> earnings record -> proof-of-work passport -> next opportunity`

## Public message hierarchy

1. **Experience should not be the thing that keeps you from getting experience.**
2. Nala helps young people earn first income through safe, structured work for real businesses.
3. Every completed task becomes verified proof that the worker can be trusted with more.
4. Businesses delegate defined outcomes without hiring for a full-time role.
5. Clear scope, fair payment, verification and controlled access are non-negotiable.

## Promise hierarchy

- **Master:** Real work. Real proof. Real progress.
- **Worker:** Earn the first income. Build the first proof. Unlock the next opportunity.
- **Business:** Get important small work done without hiring for a full-time role.
- **Trust:** Clear scope. Fair payment. Verified work.
